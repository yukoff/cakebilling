Устанавливаем **ppp**:
```
root@cake# USE="radius" emerge ppp
```

**Note:**
Необходим именно **ppp** версии **2.4.2 и выше** в нем есть **RADIUS** плагин. Флаг radius включает это плагин. Ранее он был включен по умолчанию.

Устанавливаем **pptpd**:

```
root@cake# emerge pptpd
```
